# Customer-Information-Management
## 项目介绍
使用Servlet+JSP+SQL Server+Layui+intellij idea编写的客户信息管理系统

## 相关工具
在项目文件中有相关的说明文档，请自行下载查看

## 实现功能
- 渲染sqlserver中的客户信息到前端
- 实现了查询，添加，删除和搜索功能
- 实现了分页查看功能
- 新增：使用前端框架Layui提高视觉效果
